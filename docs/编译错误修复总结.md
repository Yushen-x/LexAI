## 修复内容

### 1. 导入缺失的类

添加了两个缺失的导入：
```java
import com.lexai.backend.application.dto.request.ContractDraftRequest;
import com.lexai.backend.application.dto.response.ContractDraftResponse;
```

### 2. 实现 `draftContract()` 方法

添加了完整的 `draftContract()` 方法实现，包括：

**方法签名：**
```java
@Override
public ContractDraftResponse draftContract(ContractDraftRequest request)
```

**关键功能：**
- 接收用户的合同基本信息（名称、类型、甲乙方、金额、期限、需求）
- 先尝试调用腾讯大模型生成合同
- 若大模型失败，回退到模板生成（与 MockLegalReasoningGateway 逻辑一致）
- 返回完整的 `ContractDraftResponse`（含合同内容、摘要、生成时间）

### 3. 添加辅助方法

| 方法名 | 功能 |
|--------|------|
| `buildDraftUserPrompt()` | 构建发送给大模型的 prompt |
| `tryParseDraftResponse()` | 解析大模型的响应 |
| `generateSummary()` | 生成合同摘要 |
| `generateContractTemplate()` | 生成合同模板（兜底） |
| `convertToChineseCurrency()` | 将数字转换为中文货币单位 |

### 4. 添加 Prompt 常量


9. 第六条 - 违约责任
10. 第七条 - 争议解决
11. 第八条 - 其他条款
12. 签署区域（甲乙方签名处、日期）
